 📊 Stock Portfolio Tracker (Python)

This is a simple Stock Portfolio Tracker developed using Python.  
It allows users to enter stock names and quantities, then calculates the total investment value.

---

## 🚀 Features
- User input for stock name and quantity
- Predefined stock prices
- Calculates total investment
- Saves data in CSV file
- Simple and easy to use

---

## 🛠️ Technologies Used
- Python
- CSV module

---

## 📌 How to Run

1. Make sure Python is installed  
2. Download or clone this repository  
3. Open terminal / command prompt  
4. Run the file
